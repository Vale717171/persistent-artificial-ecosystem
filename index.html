About This World

This abstract world evolves every 6 hours. GitHub Actions runs the simulation, data/world.json stores the current state, and the UI fossil record lists retained extinctions. Git commit history is the deeper fossil layer. The system is autonomous, not user-controlled, and is a toy ecosystem rather than a biological model.
